# AI Bubble Health Dashboard - Design Specification

## Visual Identity

### Color Palette
- **Primary**: Deep charcoal (#1a1a1a) background with crisp white (#ffffff) text
- **Accent**: Electric blue (#00d4ff) for data highlights and interactive elements
- **Warning**: Amber (#ffb800) for medium risk indicators
- **Danger**: Coral red (#ff6b6b) for high risk alerts
- **Success**: Mint green (#4ecdc4) for healthy indicators
- **Neutral**: Steel gray (#6c757d) for secondary information

### Typography
- **Display Font**: "Inter" - Clean, modern sans-serif for headers and metrics
- **Body Font**: "JetBrains Mono" - Monospace for precise financial data alignment
- **Accent Font**: "Inter" - Consistent typography throughout

### Visual Language
- **Minimalist Financial**: Clean, data-first approach with surgical precision
- **High Contrast**: Ensures readability in trading environments
- **Geometric Precision**: Sharp edges, perfect alignment, mathematical clarity
- **Information Hierarchy**: Size, color, and position guide attention naturally

## Visual Effects & Styling

### Used Libraries
- **ECharts.js**: Primary data visualization with custom themes
- **Anime.js**: Smooth micro-interactions and number animations
- **Shader-park**: Subtle background particle effects for depth
- **p5.js**: Dynamic data visualization and creative coding elements
- **Matter.js**: Physics-based bubble risk visualization
- **Pixi.js**: High-performance graphics rendering
- **Splide.js**: Smooth carousel for ticker data

### Animation & Effects
- **Metric Counters**: Animated number counting for live score updates
- **Risk Gauge**: Smooth arc animations with color transitions
- **Data Points**: Subtle pulse effects on hover
- **Background**: Gentle particle flow suggesting market dynamics
- **Transitions**: Smooth page transitions with slide effects

### Header Effect
- **Liquid Metal Displacement**: Subtle animated background using shader effects
- **Depth Layering**: Multiple visual planes creating sophisticated depth
- **Color Cycling**: Gentle hue shifts reflecting market sentiment

### Data Visualization Style
- **Monochromatic Base**: Dark theme with strategic color accents
- **High Contrast Elements**: Clear visual separation between data series
- **Geometric Shapes**: Clean lines, precise angles, mathematical beauty
- **Interactive Hover States**: Detailed tooltips and data point highlighting
- **Responsive Scaling**: Adapts to different screen sizes while maintaining clarity

### Risk Visualization
- **Bubble Metaphor**: Literal bubble representations for risk levels
- **Color Coding**: Intuitive risk level indication through color
- **Size Scaling**: Visual weight corresponds to risk magnitude
- **Animation States**: Dynamic movement suggesting market volatility
